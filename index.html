# Website bán rèm cửa Đây là website bán rèm cửa đơn giản được xây dựng bằng HTML để giới thiệu sản phẩm. ## Tính năng - Hiển thị sản phẩm rèm cửa - Giao diện đơn giản, dễ sử dụng - Có thể chỉnh sửa nội dung sản phẩm ## Hướng dẫn sử dụng 1. Tải project về máy 2. Mở file `banremcua.html` bằng trình duyệt 3. Website sẽ hiển thị nội dung bán rèm cửa ## Ghi chú - Bạn có thể thay đổi tên sản phẩm, giá và hình ảnh trong file HTML - Phù hợp cho người mới học ## Tác giả Pham Van Nam
